About the App

Steam Gauge is a Python-based web app driven by Flask. It makes use of Steam's Web API, Big Picture API, and several custom Python scripts that scrape data from the Steam Store pages (if necessary). That metadata is stored in a database for easy retrieval by the Steam Gauge app. Steam Gauge does not currently store any information about your account (your login id is stored in your browser as a cookie).
